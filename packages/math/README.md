# @lunaterra/math

Math primitives for 2D geometry and transforms in Luna-Terra.

## Install

```bash
pnpm add @lunaterra/math
```

## Highlights

- `V2` vector math.
- `M3` 2D transform matrices.
- Rectangle, angle, and number utilities.

## Build from source

```bash
pnpm --filter @lunaterra/math run build
```
