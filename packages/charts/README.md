# @lunaterra/charts

Chart primitives and composition helpers for Luna-Terra.

## Install

```bash
pnpm add @lunaterra/charts
```

## Highlights

- Axis, line-series, function-plot, and stacked-area components.
- Crosshair and gesture-layer support for interactive chart scenes.
- Window tick helpers for time and viewport-aware axes.

## Build from source

```bash
pnpm --filter @lunaterra/charts run build
```
